# dot-files

## What's this

karia's dot-files!!!

## Usage

```
cd ~
git clone https://github.com/karia/dot-files.git

ln -s ~/dot-files/zsh/.zsh* .
ln -s ~/dot-files/git/.* .
ln -s ~/dot-files/tmux/.* .
ln -s ~/dot-files/vim/.* .
ln -s ~/dot-files/vimperator/.* .
```
